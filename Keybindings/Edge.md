## Keys
`C` - Control
`S` - Shift
`M` - Alt

---

Binding|Function
:-----:|:------:
`C-S-b`|Toggle bookmarks bar
`M-S-b`|Focus first item in bar
`C-S-d`|Save all open tabs as bookmarks in one folder
`C-S-e`|Open sidebar search
`M-d`|Edit search url
`M-e`|Settings menu
`C-k`|Search query in address bar
`C-t`|New tab
`C-n`|New window
`C-w`|Close tab
`C-S-w`|Close window
`C-1...8`|Switch tabs
`C-9`|Last tab
`C-tab`|Next tab
`C-S-tab`|Prev tab
`M-⬅️`|Back in history
`M-➡️`|Forward in history
